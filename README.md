"# programmable-wiki"

# Clone
`git clone https://github.com/pnaveen10/programmable-wiki.git`

# React setup
`cd programmable-wiki\react-ui`

`npm install`

# React dev build
`Programmable-wiki\react-ui\node_modules\.bin\webpack -dw`

# React prod build
`npm run build`

# Node setup
`cd programmable-wiki\backend`

`npm install`

# Start node server
`cd programmable-wiki\backend`

`node server.js`
